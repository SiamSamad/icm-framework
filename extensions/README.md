# Extensions

Optional bolt-ons for the baseline ICM shell. The core pipeline runs without any of these.

- **`adding-mcps.md`** — wire live tool integrations (ticket systems, wikis, code hosts,
  browser automation, test management) into the stages. Opt-in.
- **`adding-rules.md`** — add product configs, standing rules, and selector overrides.
- **`html-output.md`** — planned HTML run-report feature; design + integration point.
